Your Geometric Explanation (4.3):

A matrix stretches a vector when it changes the vector's length, and it may also change its direction. Eigenvectors are special directions that do not rotate under the transformation; the matrix only scales them by the corresponding eigenvalue. If the eigenvalue is greater than 1, the vector is stretched; if it is between 0 and 1, it is squished; if it is negative, it is also flipped.

SVD → PCA Connection (4.4):

  • U: left singular vectors (columns of U) are orthonormal directions in the row space; when X is data (rows = samples), the product U @ np.diag(S) gives the sample scores along components.
  • S: singular values (vector S) measure the amount of variance captured by each component; larger S → more variance.
  • Vt: rows of Vt (or columns of V) are the principal directions (loadings). For centered data, the columns of V are the eigenvectors of X.T @ X, i.e., PCA directions.

In short: SVD factorizes X = U S Vt. For PCA on centered data, the right singular vectors (V) are the principal components (directions of maximum variance), the singular values relate to explained variance (squared singular values ∝ variances), and the scores (U @ S) are the projections of samples onto those components.

def compute_psi(expected, actual, bins=10):
    """Compute Population Stability Index between two distributions."""
    expected = np.asarray(expected)
    actual = np.asarray(actual)

    breakpoints = np.linspace(np.min(expected), np.max(expected), bins + 1)
    expected_hist, _ = np.histogram(expected, bins=breakpoints)
    actual_hist, _ = np.histogram(actual, bins=breakpoints)

    expected_pct = expected_hist / len(expected)
    actual_pct = actual_hist / len(actual)

    epsilon = 1e-10
    expected_pct = np.where(expected_pct == 0, epsilon, expected_pct)
    actual_pct = np.where(actual_pct == 0, epsilon, actual_pct)

    psi = np.sum((actual_pct - expected_pct) * np.log(actual_pct / expected_pct))
    return float(psi)

np.random.seed(1)
train_dist = np.random.normal(50, 10, 1000)    # training distribution
drift_dist = np.random.normal(65, 12, 1000)    # shifted production distribution

psi_value = compute_psi(train_dist, drift_dist)
print(f"PSI: {psi_value:.4f}")
print(f"Shift severity: {'Major' if psi_value > 0.2 else 'Minor' if psi_value > 0.1 else 'Stable'}")

# Plot both distributions

Concept Drift vs Covariate Drift & PSI Thresholds (5.5):

  • Concept drift: The relationship between inputs and the target changes over time, so the same feature values can lead to different outcomes.
  • Covariate drift: The input feature distribution changes over time, while the target relationship stays the same.
  • PSI < 0.1: No significant shift; the distribution is stable.
  • PSI 0.1–0.2: Minor to moderate shift; monitor closely.
  • PSI > 0.2: Major shift; the distribution has changed enough to warrant action.
  • Retraining trigger example: Retrain when PSI stays above 0.2 for several monitoring windows or when model performance on live data drops below an agreed threshold.

p_spam = 0.30
p_free_given_spam = 0.80
p_free_given_ham = 0.05
p_ham = 1 - p_spam

# P(free) by the law of total probability
p_free = (p_free_given_spam * p_spam) + (p_free_given_ham * p_ham)

# Bayes' theorem
p_spam_given_free = (p_free_given_spam * p_spam) / p_free

print(f"P('free'): {p_free:.4f}")
print(f"P(Spam | 'free'): {p_spam_given_free:.4f}")

def naive_bayes_predict(prior_spam, p_word_given_spam, p_word_given_ham):
    p_ham = 1 - prior_spam
    evidence = (p_word_given_spam * prior_spam) + (p_word_given_ham * p_ham)
    return (p_word_given_spam * prior_spam) / evidence

np.random.seed(7)

CLT Reflection (6.4):

The CLT matters because it explains why averages and sample means often behave in a nearly normal way, even when the original data is not normal. That is useful in statistics and machine learning because it lets us use normal-based confidence intervals, hypothesis tests, and error estimates with much more confidence. The CLT justifies the assumption that sampling distributions of many estimators are approximately normal when the sample size is large enough.
